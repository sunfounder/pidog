3-pin 电池
=================

.. image:: img/3pin_battery.jpg

* **VCC**：电池正极，此处有两组 VCC 和 GND，目的是增大电流并减小电阻。
* **中间引脚**：用于平衡两个电芯之间的电压，从而保护电池。
* **GND**：电池负极。

这是 SunFounder 定制的电池组，由两节 18650 电池组成，容量为 2000mAh。连接器为 XH2.54 3P，插入 Robot HAT 后即可直接充电。

**特性**

* 成分：锂离子
* 电池容量：2000mAh，14.8Wh
* 电池重量：90.8g
* 电芯数量：2
* 连接器：XH2.54 3P
* 过放保护：6.0V
