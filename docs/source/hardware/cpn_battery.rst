
5-pin 电池
=================

.. image:: img/2battery.jpg
    :width: 300
    :align: center

* **VCC**：电池正极，这里有两组 VCC 和 GND 是为了增加电流和降低电阻。
* **Middle**：用于平衡两节电池之间的电压，从而保护电池。
* **GND**：电池负极。

这是 SunFounder 制作的定制电池组，由两节 18650 电池组成，容量为 2000mAh。连接器为 PH2.0-5P，插入 Robot HAT 后可直接充电。

**特点**

* **组成**：锂离子
* **电池容量**：2000mAh, 14.8Wh
* **电池重量**：90.8g
* **电节数量**：2
* **连接器**：PH2.0-5P
* **过放保护**：6.0V
