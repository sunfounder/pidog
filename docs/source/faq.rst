P1: Sobre el error "pinctrl: not found".
-------------------------------------------------------------------

Si encuentras el siguiente error:

.. code-block::

    pinctrl: not found

Esto indica que has instalado el sistema Bullseye. Se recomienda instalar el sistema **Bookworm** en su lugar.
