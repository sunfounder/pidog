Robot HAT
==================

**V2 版本：**

|link_robot_hat_v5| 是一款多功能扩展板，可让 Raspberry Pi 快速变身为一台机器人。
V2 Robot HAT 在标准版架构的基础上集成了麦克风，支持音频录制和语音识别功能。

详细说明请参考：|link_robot_hat_v5|。


**标准版本：**

|link_robot_hat_v4| 是一款多功能扩展板，可让 Raspberry Pi 快速变身为一台机器人。
板上集成了 MCU，为 Raspberry Pi 扩展了 PWM 输出和 ADC 输入，
同时还配备了电机驱动芯片、I2S 音频模块和单声道扬声器。
此外，还引出了 Raspberry Pi 本身的 GPIO。

它还配备了一个扬声器，
可用于播放背景音乐、音效以及实现 TTS 功能，让你的项目更加有趣。

支持 7-12V 电源输入，带有 2 个电池指示灯、1 个充电指示灯和 1 个电源指示灯。
板上还有一个用户可用 LED 和一个按钮，方便你快速测试一些效果。

详细说明请参考：|link_robot_hat_v4|。
