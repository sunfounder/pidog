# trot

**File:** `pidog/trot.py`

## Module Description

A full MOVE divided into 8 SECTIONs. like below:
leg| 0 | 1 |
====|===|===|
 1  |^^^|___|
 2  |___|^^^|
 3  |___|^^^|
 4  |^^^|___|

4 legs move in this order: 1, 4, 2, 3
there will be a break after every leg move

every SECTION devide into 4 STEPs
