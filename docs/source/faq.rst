Q1: Riguardo all'errore "pinctrl: not found".
-------------------------------------------------------------------

Se incontri l'errore seguente:

.. code-block::

    pinctrl: not found

Significa che hai installato il sistema Bullseye. Si consiglia di installare il sistema **Bookworm** al suo posto.
