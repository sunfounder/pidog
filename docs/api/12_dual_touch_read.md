**File:** `basic_examples/12_dual_touch_read.py`

## Module Description

read dual_touch module

API:
    Pidog.dual_touch.read()
        - return str, dual_touch status:
            - 'N'  no touch
            - 'L'  left touched
            - 'LS' left slide
            - 'R'  right touched
            - 'RS' right slide
