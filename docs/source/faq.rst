Q1: Zum Fehler "pinctrl: not found"
-------------------------------------------------------------------

Wenn du den Fehler siehst:

.. code-block::

    pinctrl: not found

bedeutet das, dass du das Bullseye-System installiert hast. Es wird empfohlen, stattdessen das **Bookworm-System** zu installieren.
