玩转 Python
====================

如果您希望使用 Python 进行编程，  
那么您需要掌握一些基础的 Python 编程技能以及 Raspberry Pi 的基础知识，  
请首先根据 :ref:`quick_guide_python` 配置好您的 Raspberry Pi。
