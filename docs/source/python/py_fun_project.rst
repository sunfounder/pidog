3. 趣味Python 项目  
=========================

在这里，我们深入探讨一系列激动人心的项目，展示了 PiDog 的多功能性和强大能力。  
从 :ref:`py_wake_up` 中设置唤醒例程的基础，到 :ref:`py_ball_track` 中球体追踪的高级动态，每个项目都为你提供了探索机器人编程世界的独特视角。无论你是希望让 PiDog 巡逻、响应指令、做俯卧撑，还是按命令嚎叫，都有适合你的项目。此外，对于那些希望将 PiDog 的功能扩展到计算机接口的用户，我们还提供了键盘和应用程序控制的教程。快来加入我们，带着这些动手实践的 Python 项目，开始一段发现与乐趣并存的 PiDog 编程之旅吧！
