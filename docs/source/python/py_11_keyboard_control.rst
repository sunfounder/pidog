
11. 使用键盘操控 PiDog
======================================

**运行代码**

.. raw:: html

    <run></run>

.. code-block::

    cd ~/pidog/examples
    sudo python3 11_keyboard_control.py

运行程序后，您将在终端中看到一张按键说明图。现在，您可以直接通过键盘在终端中操控 PiDog 了。
