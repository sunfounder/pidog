FAQ
===========================

Q1: "pinctrl: not found"エラーについて
-------------------------------------------------------------------

次のエラーが表示された場合：

.. code-block::

    pinctrl: not found

これは、Bullseyeシステムがインストールされていることを意味します。代わりに **Bookwormシステム** をインストールすることをお勧めします。
