3-pin 电池
=================

.. image:: img/3pin_battery.jpg

* **VCC**：电池正极。这里有两组 VCC 和 GND，用于增加电流、降低电阻。  
* **Middle（中间引脚）**：用于平衡两节电芯间的电压，从而保护电池。  
* **GND**：电池负极。

这是一款由 SunFounder 定制的电池包，由两节 18650 电池组成，容量为 2000mAh，接口类型为 XH2.54 3P，可直接插入 Robot HAT 充电使用。

**产品特性**

* 电池类型：锂离子（Li-ion）  
* 容量：2000mAh（14.8Wh）  
* 重量：90.8g  
* 电芯数：2 节  
* 接口：XH2.54 3P  
* 过放保护电压：6.0V
