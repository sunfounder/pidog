11通道光板
=============================

.. image:: img/cpn_lightboard.png
   :width: 500
   :align: center


这是一个11通道RGB LED模块，配备了11个由SLED1735芯片控制的RGB LED。

SLED1734可以驱动多达256个LED和75个RGB LED。
在由SLED1734控制的LED阵列中，每个LED都具有开/关、闪烁、呼吸灯以及自动同步等多种功能。
该芯片内置了PWM（脉冲宽度调制）技术，可提供256级亮度调节。它还具有16级点校正功能。



**规格**

* 电源供应：3.3V
* 通讯方式：IIC
* 连接器：SH1.0 4P
* LED类型：3535 RGB LED

**管脚定义**

* GND - 地线输入
* VCC - 电源输入
* SDA - IIC SDA
* SCL - IIC SCL
